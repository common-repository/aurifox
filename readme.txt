== Installation ==

**Easy Installation**: In your WordPress dashboard go to Plugins > Add New and search for "aurifox" and you will be able to quickly install the plugin.

**Manual Installation**: Download the aurifox WordPress plugin .zip file. Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation via a FTP and then activate the Plugin from Plugins page.

**How to Setup Authenication Key**

Login to your aurifox account and go to your Account details and to get the
authentication key *WordPress API key*, which is unique to your account.

You will also need the email address you signed up with. From there you can use the email and key to connect the plugin to your account.
